# 고용드림 홈페이지(웹용) 시안 안내

> 검색노출(네이버·구글)용 **멀티페이지 홈페이지 시안**입니다.
> 아임웹·카페24 어느 쪽이든 옮길 수 있도록 특정 플랫폼에 묶이지 않게 제작했습니다.
> (휴대폰 첨부·발송용은 별도 파일 `goyongdream_landing_v2_mobile.html` 입니다.)

---

## 1. 페이지 구성 (클릭되는 시안)

| 파일 | 메뉴 | 내용 |
|---|---|---|
| `index.html` | 홈 | 히어로 · 협업구조 · 핵심지원금 · 사례·공지 미리보기 · CTA |
| `about.html` | 회사소개 | 미션 · 노무×세무 협업 · 4가지 원칙 |
| `service.html` | 서비스 안내 | 고용지원금(노무) · 세액공제·경정청구(세무) · 진행 4단계 (드롭다운 연결) |
| `cases.html` | 환급 사례 | 업종별 환급 사례 6건 |
| `notice.html` | 공지·블로그 | 게시판 목록 (글 클릭 시 상세로) |
| `post.html` | (글 상세) | 게시글 상세 — 목록에서 **들어가는** 구조 시연 |
| `apply.html` | 무료 진단 | 신청 폼 → 구글시트 웹훅 → taxpartner.kr 진단 연결 |

폴더: `css/style.css`(공통 디자인), `js/site.js`(모바일 메뉴·활성표시)

> **확인 방법**: `index.html` 을 더블클릭 → 상단 메뉴로 페이지 이동, 서비스 드롭다운, 공지→글 들어가기, 모바일 햄버거 메뉴까지 전부 동작합니다.

## 2. 게시 전 채워야 할 자리표시자

| 위치 | 현재 값 | 바꿀 내용 |
|---|---|---|
| 전체 푸터 · apply 폼 | `000-0000-0000` | 고용드림 대표 연락처 |
| `apply.html` 스크립트 | `여기에_고용드림_새_웹훅_URL_붙여넣기` | 고용드림 구글시트 Apps Script 웹훅 URL |
| (선택) 협업 섹션 | "전문 노무법인 / 세무법인" | 실제 제휴 법인명 (확정 시) |

> 웹용 폼의 유입태그는 `goyongeum_dream_web` 로 설정되어, 모바일 랜딩(`goyongeum_dream`)과 시트에서 구분됩니다.

## 3. 실제 운영(검색노출)으로 가는 방법 — 두 갈래

### 경로 A. 아임웹/카페24 안에 올리기 (검색노출에 유리)
- 이 시안의 **디자인·문구·구조를 기준**으로, 각 플랫폼의 페이지/메뉴 기능으로 페이지를 1:1로 만듭니다.
- 메뉴(홈·회사소개·서비스·사례·공지·진단)를 **각각 실제 페이지(별도 URL)** 로 만들어야 네이버·구글에 페이지별로 잡힙니다.
- 공지·블로그 게시판, 모바일 자동대응, sitemap·검색엔진 연동은 **플랫폼 기본 기능**을 사용합니다.

### 경로 B. 이 시안을 그대로 호스팅
- 이 폴더(`goyongdream_website`)를 웹호스팅에 올리고 **도메인을 연결**하면 그대로 사이트가 됩니다.
- 이후 **네이버 서치어드바이저 / 구글 서치콘솔**에 사이트 등록 → 색인 요청을 하면 검색에 노출됩니다.
- 게시판 글쓰기·회원관리 같은 동적 기능이 필요하면 그땐 플랫폼(아임웹/카페24)이 더 적합합니다.

- 모바일 랜딩과 동일한 **딥네이비 + 골드** 브랜드로 통일했습니다.
