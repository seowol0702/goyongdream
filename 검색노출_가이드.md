# 고용드림 검색 노출(네이버·구글) 가이드

도메인 연결까지 끝났으니, 이제 검색엔진에 **직접 등록**하면 됩니다. (자동으로는 안 잡힙니다.)

---

## 0. 먼저 — sitemap·robots 업로드
방금 만든 두 파일을 GitHub 저장소 **루트(index.html과 같은 위치)**에 올리세요.
- `sitemap.xml`
- `robots.txt`

올린 뒤 브라우저에서 `https://goyongdream.kr/sitemap.xml` 가 열리는지 확인.

---

## 1. 구글 등록 — Google Search Console
1. https://search.google.com/search-console 접속 → 로그인
2. **속성 추가 → "URL 접두어"** 선택 → `https://goyongdream.kr` 입력
3. **소유권 확인** — "HTML 태그" 방식 추천
   - 구글이 주는 `<meta name="google-site-verification" content="...">` 한 줄을 복사
   - → 그 줄을 **저에게 주시면 모든 페이지 `<head>`에 넣어드립니다** (또는 HTML 파일 다운로드 방식 선택 가능)
   - 넣은 파일을 GitHub에 업로드 후 구글에서 "확인" 클릭
4. 확인되면 좌측 **Sitemaps** → `sitemap.xml` 입력 → 제출
5. 상단 검색창에 `https://goyongdream.kr` 입력 → **색인 생성 요청**

---

## 2. 네이버 등록 — 네이버 서치어드바이저
1. https://searchadvisor.naver.com 접속 → 로그인
2. **웹마스터 도구 → 사이트 등록** → `https://goyongdream.kr` 입력
3. **소유확인** — "HTML 태그" 방식 추천 (구글과 동일하게 메타태그 한 줄)
   - `<meta name="naver-site-verification" content="...">` → 저에게 주시면 `<head>`에 넣어드립니다
4. 확인되면 **요청 → 사이트맵 제출** → `https://goyongdream.kr/sitemap.xml`
5. **요청 → 웹페이지 수집** 에 주요 주소(`https://goyongdream.kr/` 등) 넣어 수집 요청

---

## 3. 등록 후
- 실제 검색결과 노출까지 **며칠 ~ 2~3주** 걸립니다 (신규 사이트는 더 걸릴 수 있음).
- `site:goyongdream.kr` 를 네이버·구글에 검색하면 색인된 페이지를 확인할 수 있습니다.

## 4. 더 빨리·더 많이 노출되려면
- **공지·블로그에 글을 꾸준히** 올리세요. (지원금 정보, 사례 등 — 검색 유입의 핵심)
- 글 제목·본문에 사람들이 실제로 검색하는 표현 사용 (예: "청년일자리도약장려금 신청", "통합고용세액공제 경정청구").
- 네이버는 **네이버 블로그/플레이스(지도) 등록**을 함께 하면 노출에 유리합니다.

---

### 저에게 주시면 바로 해드리는 것
- 구글/네이버 **소유확인 메타태그**(content 코드) → 전 페이지 head에 삽입
- 추가 블로그 글 작성 (검색 잘 되도록 제목·키워드 포함)
